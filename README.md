张量：可以直接理解为多维数组。


矩阵：在编程当中可以理解为二维数组

1.矩阵加法：

[1 2]  +  [3 4] =  [1+3 2+4] =  [4 6]
[3 4] + [5 6] = [3+5 4+6] = [8 10]


2.矩阵乘法：

[1 2] × [3 4] = [1×3+2×5 1×4+2×6] = [13 16]
[3 4] × [5 6] = [3×3+4×5 3×4+4×6] = [29 36]


3.矩阵转置：

把矩阵的行和列"交换位置"，原来的第i行第j列的元素，转置后就变成了第j行第i列的元素。

原始矩阵A（2行3列）：
[1 2 3]
[4 5 6]

转置后的矩阵A^T（3行2列）：
[1 4]
[2 5]
[3 6]
